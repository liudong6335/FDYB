## 性格 → 行为快查

侵略高 + 胆怯低 = 冲脸不回头（Rookie / Berserker）
侵略低 + 胆怯高 = 怂着不打（Scholar / Coward）
自保高 + 侵略低 = 秒跑（Coward / Scholar）
支援高 + 侵略中 = 协同攻击（Guardian / Veteran）
胜利高 + 侵略中 = 目标导向（Runner / Guardian）
独立高 + 自保中 = 个人主义（Hunter / Cunning）
贪婪高 = 到处捡东西（Hunter）
